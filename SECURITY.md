# Security Policy

## Supported Versions

We only support the latest version of **Resile**. Please ensure you are running the most recent release to receive the latest security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please do **not** open a public issue. Instead, please report it through one of the following channels:

1.  **GitHub Security Advisory**: If available on this repository, please use the "Report a vulnerability" button in the "Security" tab.
2.  **Direct Contact**: [Insert your preferred contact method or email if desired, otherwise refer to GitHub private reporting].

### Our Process

- We will acknowledge your report within 48 hours.
- We will provide a detailed response with a timeline for a fix.
- We will coordinate a public disclosure once the fix is released.

Thank you for helping keep this project safe!
