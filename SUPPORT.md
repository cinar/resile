# Support

## How to Get Help

Thank you for using **Resile**! If you need help, please follow these steps:

1.  **Check the Documentation**: Read the `README.md` and the [Architectural Specification](docs/SPEC.md).
2.  **Search Existing Issues**: Your question or bug might have already been addressed.
3.  **Open an Issue**: If you've found a bug or have a feature request, please [open a new issue](https://github.com/cinar/resile/issues).

## Community

Please be respectful and helpful to other community members. All participation is governed by our [Code of Conduct](CODE_OF_CONDUCT.md).

## Security

If you have found a security vulnerability, please refer to our [Security Policy](SECURITY.md).
